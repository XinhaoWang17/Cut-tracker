## 项目背景

### 仓库
- GitHub: https://github.com/XinhaoWang17/Cut-tracker
- 部署：GitHub Pages（index.html 为唯一入口）

### 技术架构
- 单文件 PWA（`index.html`）
- 无依赖，纯 HTML + CSS + JavaScript
- 数据存储：localStorage（key 前缀 `xh-`）
- 支持 iOS Safari "添加到主屏幕" 离线使用
- 响应式，最大宽度 480px，针对手机优化

### 核心功能
- 训练日记录（每组打勾 + 重量记录）
- 周视图、10/12周进度条
- 身体数据记录（体重、腰围、体脂）
- 精简/完整模式切换
- 深色/浅色主题自动跟随系统

### 开发注意事项
- 不能引入外部框架或构建工具
- 所有训练数据硬编码在 `EXERCISES` 对象中
- `PROGRAM_START` 和 `TOTAL_WEEKS` 控制周期
- 训练日周排布由 `WEEK_TPL` 控制
- 日志 key 格式：`YYYY-MM-DD-dN`（N 为 dayId）
